
# TeamMusic PC Client

## How to use?

- Download nodejs.

- Clone the git repository

- Change your directory to teammusic-pc

- Run `npm install`

- Run `yarn`

- Run `npm start` to start the electron application

## How to Build the app?
### Windows :
    npm run win-dist
    
### Linux :
    npm run linux-dist

### Mac OS :
    npm run mac-dist
    
### Run your build :
    npm run dist
